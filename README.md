# Cypress Automation Framework

A robust, enterprise-grade end-to-end test automation framework built with Cypress and JavaScript, implementing Page Object Model (POM) architecture for scalable and maintainable test suites.

## Tech Stack

- **Framework:** Cypress
- **Language:** JavaScript
- **Architecture:** Page Object Model (POM)
- **API Testing:** Cypress intercept and REST API validation
- **CI/CD:** GitHub Actions
- **Test Data:** Fixtures

## Project Structure

cypress/
├── pages/          # Page Object Model classes
├── scripts/        # Test spec files
├── fixtures/       # Test data files
└── support/        # Custom commands and configuration

## Features

- End-to-end UI automation
- API request validation and interception
- Reusable Page Object Model components
- Data-driven testing using fixtures
- Clean and maintainable test architecture

## Getting Started

### Prerequisites
- Node.js v14 or higher
- npm

### Installation

git clone https://github.com/riyazuddinsyed/cypress-automation-framework.git
cd cypress-automation-framework
npm install

### Running Tests

npm run cypress:open
npm run cypress:run
